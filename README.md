# Suno Prompt Maker

Suno AI 向けのプロンプトメーカー。ブラウザで動く静的 Web アプリです。

## 特徴

- **35カテゴリ**の音楽要素をランダムに組み合わせてプロンプトを自動生成
- **64テンプレート**（プロンプト64種 + 歌詞構造15種）
- **重み付きラウンドロビン**でカテゴリごとの出現割合を制御
- **履歴管理**で過去のプロンプトを保存・呼び出し
- **ダーク/ライトモード**対応
- **モバイル対応**（レスポンシブデザイン）
- **ツールチップ**でカテゴリ説明とサンプル候補を表示
- **ラベルクリック**で0%⇔100%切替

## テンプレート

### プロンプトテンプレート（64種）
- 王道ポップス、バラード、エレクトロダンス、シティポップ
- ロック、ジャズ、シンセウェーブ、フォーク
- K-Pop、ボカロ、ゲームBGM、オーケストラ
- ヒップホップ、レゲエ、ラテン、ワールドミュージック
- メタル、パンク、アンビエント 等

### 歌詞構造テンプレート（15種）
- 王道J-POP、アニメOP、バラード
- EDM / Future Bass、Kawaii Future Bass
- ロック、メタル、Hip-Hop / Rap
- City Pop、Synthwave、Vocaloid

- ミュージカル風

## 技術スタック

- HTML5 / CSS3 / JavaScript (vanilla)
- Tailwind CSS (CDN版)
- サーバー不要（クライアントサイド完結）
- データ: `data.js` に集約

## 使い方

1. ブラウザで `index.html` を開く
2. 「メインプロンプト」に追加の指示を入力（任意）
3. 「要素の割合」でカテゴリごとの出現率を調整
4. 「プロンプトを生成」をクリック
5. 生成されたプロンプトをコピーして Suno に貼り付け

### ショートカット
- **ラベルクリック**: 0% ⇔ 100% に切替
- **テンプレート**: クリックでコピー + 自動適用

## デプロイ

```bash
# FTPデプロイ（事前に deploy-ftp.sh を編集）
./deploy-ftp.sh

# ドライラン（確認のみ）
./deploy-ftp.sh --dry-run
```

## カテゴリ追加方法

`data.js` の2箇所を編集するだけ:

```js
// 1. CATEGORIES に追加
newCategory: {
  name: 'カテゴリ名',
  label: 'スライダー表示名',
  desc: '説明文',
  dataKey: 'newCategory',
  defaultWeight: 0,
},

// 2. PROMPT_DATA に配列追加
newCategory: ['item1', 'item2', ...],
```
