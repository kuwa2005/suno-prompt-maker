## 決め事・ルール

### 応答
- 応答は日本語で

### デザイン
- 確認ダイアログのボタン順は「はい」→「いいえ」（同意/非同意の順）
- Tailwind CSS CDN版を使用（CLI移行はレイアウト崩壊のため撤回済み）
- ダーク/ライトモード対応

### デプロイ
- FTPデプロイ: `./deploy-ftp.sh`（b45.coreserver.jp）
- デプロイ前に `node --check *.js` で構文チェック実行
- data.js の `enka` と `enkaMusic` プロパティが正しくデプロイされているか確認

### カテゴリ管理
- カテゴリ追加は `data.js` の2箇所（CATEGORIES + PROMPT_DATA）
- `CATEGORIES` の `dataKey` は `PROMPT_DATA` のキーと一致させること
- 現在のカテゴリ数: 38

### AI機能
- ternlight: フォールバックモードで動作（CDN不可のため）
- Chrome Translator API: Chrome 138+ のみ対応、ユーザージェスチャーが必要
- メインプロンプトから反映時はジャンルも含める（`excludeGenre=false`）

## 技術スタック

- HTML5 / CSS3 / JavaScript (vanilla)
- Tailwind CSS (CDN版)
- ternlight (フォールバックモード)
- Chrome Translator API (オプション)

## 注意事項

1. **data.js のデプロイ確認**: `enka` と `enkaMusic` プロパティが正しくデプロイされているか確認
2. **Tailwind CSS**: CDN版を使用。CLI版はレイアウト崩壊の原因となるため使用不可
3. **ternlight**: CDN不可のためフォールバックモードで動作
4. **Playwright**: デバッグ用にセットアップ済み。`node debug.js` でコンソールログ取得可能
