このアプリについて

概要

Suno Prompt Maker は、Suno AI 向けのプロンプトメーカーです。 38カテゴリの音楽要素を組み合わせてプロンプトを生成できます。 すべての処理はブラウザ内で完結する静的Webアプリであり、サーバー不要で動作します。

主な機能

  • 38カテゴリの音楽要素（ジャンル、ムード、楽器など）
  • メインプロンプト + 要素の割合（スライダー）による細かい調整
  • プロンプト生成（ランダム）/ プロンプト生成(AI)
  • (AI) メインプロンプトから反映 — メインプロンプトの内容に基づきスライダーを自動設定
  • セマンティック検索 — 各カテゴリの候補を意味ベースで絞り込み
  • 履歴の意味ベース検索
  • 64テンプレート（プロンプト64種 + 歌詞構造15種）
  • 重み付きラウンドロビンでカテゴリごとの出現割合を制御
  • 履歴管理で過去のプロンプトを保存・呼び出し
  • ダーク/ライトモード対応
  • モバイルレスポンシブ対応

プライバシーポリシー

このアプリケーションは、すべての処理をユーザーのブラウザ内で完結させています。 サーバーへのデータ送信は一切行いません。

収集する情報: このアプリは個人情報を収集しません。

ローカルストレージ: プロンプトの履歴とテーマ設定は、 ブラウザのローカルストレージに保存されます。このデータはあなたのデバイス内にのみ存在し、 外部に送信されることはありません。

Cookie: メインプロンプトの入力内容は、利便性のためにCookieに保存されます。 このCookieは365日間有効です。

AI機能: ternlight（WASM）による埋め込みベクトル生成、スライダー自動設定、 セマンティック検索、プロンプト生成(AI) はすべてブラウザ内で完結します。 入力されたテキストや生成結果が外部サーバーに送信されることはありません。 日本語表示には Chrome Translator API をオプションで利用します（対応ブラウザのみ）。

IndexedDB: AI機能で使うタグの埋め込みベクトルは IndexedDB にキャッシュされ、 次回以降の起動を高速化します。キャッシュはデバイス内にのみ保存され、外部に送信されることはありません。 データ更新後や不具合時は、下の「ローカルキャッシュ再構築」で削除・再構築できます。

ローカルキャッシュ

タグインデックスのIndexedDBキャッシュを削除し、再構築します。
