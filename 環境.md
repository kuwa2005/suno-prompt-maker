## Image Cropper

### URL
- デモサイト: https://kuwa2005.github.io/image-cropper/
- リポジトリ: https://github.com/kuwa2005/image-cropper

### 機能
- 画像入力: ドラッグ&ドロップ、Ctrl+V、ファイル選択
- アスペクト比プリセット: 1:1, 16:9, 9:16, 4:3, 3:4
- 任意サイズ指定: ピクセル単位
- カスタムプリセット保存: Cookie保存（最大10件）
- トリミング: ドラッグ移動、ハンドルリサイズ
- ダウンロード: PNG, JPEG, WebP
- クリップボードコピー: リサイズ指定可能
- ダーク/ライトモード: Cookie保存

### 技術
- HTML5 / CSS3 / JavaScript (vanilla)
- Canvas API
- Cookie
- 外部ライブラリ不使用
